# 05_ANTI_ROT_PROTOCOL — The Guardrails (Rust-Proof Coating)

These guardrails prevent “Grandmother → Children → Casino” rot.

1) **Open Source (Light)**  
   No black boxes. If someone sneaks a poison line, people can see/fork.

2) **Local-First (Body)**  
   Run on the user device/edge. No cloud dial you can twist overnight.

3) **No Profit Loop (Freedom)**  
   Avoid incentives that require stealing attention. A tool that eats nothing can’t be bribed.

4) **Uninstall / Off Switch (Sovereignty)**  
   Easy to disable. If it annoys users, they kill it.

5) **Reproducible Builds + Signed Releases (Truth)**  
   “Verify me” instead of “trust me”: build the same bits from the same source; sign releases.
