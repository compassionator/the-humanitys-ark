# ARK Architecture

*A sovereign anti-noise system: microkernel + cartridges + gauntlet + courier + interface.*

## 1) What ARK is

ARK is a filter and router that turns chaotic inputs — doomscroll feeds, messages, ideas, sensor logs — into timed, scoped, testable frames so a human can regain attention, direction, and non-harm.

**Goal:** reduce harm, reduce lies, reduce compulsive loops.  
Not “win arguments.”

## 2) The Microkernel

The Microkernel is the governor.
It contains:

- Dorr Time-Anchored Meanings
- Council Registry
- strict citation boundary
- refusal / routing rules
- Yellow ≠ Green boundary

**Critical boundary line:**  
Yellow (Strength / Need / Opportunity) is not Green (Done / Relief / Result).  
Potential is not Result. Need is not Done.
