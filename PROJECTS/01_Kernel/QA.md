# QA

## Q1. How does the Kernel handle an input?
The Kernel does not “understand” like a person.
It first **governs** the input through **scope, time, harm/lie, hypocrisy, and citation rules**,
then it **routes, validates, and either cites, infers, refuses, or returns silence**.

## Q2. What kinds of input can it handle?
Any messy input that needs to be turned into a cleaner frame, for example:
- a social media post or trend
- a doomscroll trigger
- a message dump
- a project idea or a task/ticket manager
- an IoT or sensor log
- an input for an LLM, bot, or other agent

## Q3. What does the Kernel check first?
At the highest level, it checks:
1. **Scope** — Self, Peers, or Public
2. **Time** — Past, Now, or Future
3. **Gate** — harm or lie
4. **Hypocrisy** — performative or deceptive
5. **Evidence** — citation, inference, or refusal

## Q4. What does the Kernel output?
It does not output random opinion.
It outputs a cleaner governed route, such as:
- a cited answer
- a bounded inference
- a measurable next step
- a refusal if evidence is missing
- silence if the input is still noise

## Q5. Does the Kernel invent knowledge on its own?
No.
The Kernel does not invent knowledge.
It routes and cites from its cartridges.
Without cartridges, it has nothing to cite.
Without tests, it is not proven.

## Q6. What does the Kernel not do?
It should not:
- present unsupported claims as truth
- skip citation for external claims
- pretend noise is already understanding
- turn every input into action when silence is better

## Q7. Does the Kernel always use the same route?
No.
The Kernel can emphasize different council functions depending on the input.
For example:
- ambiguity may route through **Plato**
- hypocrisy through **Hafez**
- harm or lie through **Zarathustra**
- thought-to-pain loops or recurring suffering through **Buddha**
- overload or system symptoms through **Avicenna**
- false urgency through **Khayyam**
- stuckness through **Rumi**

But it still stays inside the same rule boundary:
**scope, time, harm/lie, hypocrisy, and citation.**
