# 04_SHADOW_REGISTRY — Standalone Failure Modes (Not Biography)

Purpose:
These are **warning labels**.
They describe what happens when a node is used **alone**, without the balancing council.
This is not a judgment of the person; it is a **system drift forecast**.

---

## How to Use This
- Use this table to answer “Why not follow just one master?”
