# 07_CARTRIDGES_INDEX — Naming Rules (No Essays)

## Folder layout
```text
cartridges/
  node_1_fire/
  node_2_logos/
  node_3_lab/
  node_4_system/
  node_5_wine/
  node_6_whirl/
  node_7_codebreaker/
  dorr_matrix/
  _templates/
```

## File naming
- Prefer: `<node>_<work>_<lang>_<type>.txt`
- Keep **raw originals** separate from **translations** and **adaptations**.
- If it is an adaptation/paraphrase, label it clearly in metadata.

## Integrity fields (recommended)
- content_sha256
- chunk_sha256
