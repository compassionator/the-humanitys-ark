Simple workflow

5-minute first run

  1. 1
    Customize your Lens

    Add roles you want, related roles, deal-breakers, and preferences. Basic mode is designed for everyday use.

  2. 2
    Open LinkedIn Jobs or SEEK Jobs

    Open a real job page or supported results page in the tab you want ARK Lens to watch.

  3. 3
    Start Session

    Open the ARK Lens popup and select Start Session. The main A icon changes from gray to green while the session is active.

  4. 4
    Browse jobs in the same tab

    ARK Lens captures supported jobs locally. Use Capture Current Job if you want to save the open job immediately.

  5. 5
    Review and give feedback

    Open the report, select a row, inspect the evidence, and mark it Relevant, Not relevant, or Unsure.

Transparent ranking

What the match percentage means

The match percentage comes deterministically from the phrases and policies in your Lens. It is not a probability, an application recommendation, or an AI opinion.

Strong Match
Meets your configured high-fit threshold.
Review
Has mixed or incomplete rule evidence.
Low Match · 10.1–49.99%
Has weak evidence or meaningful penalties and may still be worth a quick review.
Ignore · 0–10%
Has no useful role-fit evidence, a decisive penalty, or a blocker.
Relevance feedback
Your separate judgment; it never silently changes the match percentage or your Lens.

When a site changes

Fix Capture

If a title, company, date, or description is missing, open Fix Capture in the popup and select Check Capture.

  1. Preview the Help File before saving it.
  2. Share it only with a trusted helper or your chosen AI.
  3. Preview and test any returned Repair File.
  4. Activate only after the live-page test passes.

Never send raw page HTML. The Help File is designed to provide safer examples and diagnostics.

Privacy

Your data stays in this browser

ARK Lens has no account, backend, analytics, or automatic AI connection. Job descriptions, notes, Lens settings, decisions, and relevance feedback remain in Chrome local extension storage unless you explicitly export a file.

The Alpha Test Summary below contains aggregate counts and version information only. It excludes job titles, companies, descriptions, URLs, notes, feedback details, Lens names, session IDs, and tab IDs.

Please expect rough edges

Known alpha limits

  • Only LinkedIn Jobs and SEEK Jobs are supported.
  • The match percentage is lexical and deterministic; this release does not use semantic AI.
  • Relevance feedback is captured for review but does not automatically retrain or rewrite your Lens.
  • Job sites can change their page structure; use Fix Capture if required fields disappear.
  • A capture session follows one supported tab and stops if that tab closes.
  • Chrome extension storage is local to this browser profile and is not a cloud backup.
