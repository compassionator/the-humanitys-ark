# IoT Hub

## Role
Tool / Seed

## Goal
Group the real-world hardware and device experiments that can be built, tested, and shipped as practical sovereign tools.

## Why it matters
This folder is the hands-on edge of the Ark: sound, sensors, off-grid power, field devices, wearables, and practical protection systems. It is where architecture touches dirt, signal, hardware, and daily life.

## Current next step
Start with the lowest-complexity, highest-proof devices first, while preserving the full concept stack in one visible place.
