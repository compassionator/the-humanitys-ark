# The Cornea

## Role
Seed / Research / Prototype

## Goal
Remove the “phone-in-hand” loop in phases by building a voice-first AI intermediary and then offloading selected tools into lightweight AR / WebXR glasses anchored to real space.

## Why it matters
The Cornea is not just about glasses. It is about reducing finger-noise, screen addiction, and flat-interface dependence by making intelligence more glanceable, spatial, and hands-free.

## Current next step
Build the first proof:
- pin a digital sticky note to a real wall or table
- prove that a simple digital frame can stay world-locked in physical space

## Status
Draft

## Core phases

### Phase 1 — Voice / AI Intermediary
Insert a voice-first layer between the user and the phone so the user needs less:
- tapping
- scrolling
- searching
- finger-driven navigation

This layer should help filter noise and retrieve only what matters.

### Phase 2 — Offloaded AR / WebXR Layer
Move selected tools out of the phone and into lightweight AR space:
- maps
- notes
- task prompts
- reminders
- hourglass / time cues
- simple memory retrieval

The goal is not “full sci-fi goggles” first.
The goal is cheap, modular, useful overlays.

## Expanded direction from older notes
The Cornea can include:

- Dorr AR prototype UI for glance-based interaction
- AI-driven voice navigation
- basic gesture recognition
- attachable HUDs for existing glasses
- 3D-printed modular AR glasses
- gaze tracking for selection
- AI memory retrieval
- digital hourglass visualization in AR
- multi-device sync with phone / tablet / desktop when needed

## Design rule
Minimal, fast, distraction-light.

Do not overwhelm the field of view.
Show only what helps the user act, remember, or orient.

## Challenges
- current AR hardware is bulky, expensive, or too closed
- real-time voice interaction must work in noisy environments
- world-locking must feel stable enough to trust
- battery and processing limits will matter
- the interface must stay minimal, not become another screen prison

## Relationship to other projects
- **Dorr** provides the structured color/task/memory logic
- **Kernel** governs meaning and boundaries
- **G-Rank** may later become one visible rating layer in AR
- **IoT Hub** may supply some of the hardware experiments
- **Nature Tech** may influence lightweight form factors and adaptive design
